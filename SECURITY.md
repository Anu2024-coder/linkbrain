# Security Policy

LinkBrain interacts with physical hardware and execution systems. Security issues are treated seriously.

---

## Reporting vulnerabilities

If you discover a security vulnerability, please **do not open a public issue**.

Instead, report it responsibly by contacting the project maintainers with:

* A clear description of the issue
* Steps to reproduce (if applicable)
* Potential impact

We will review and respond as quickly as possible.

---

## Scope

Security concerns may include:

* Unauthorized device control
* Bypassing validation or execution contracts
* Unsafe hardware behavior triggered by AI output
* Credential or key leakage

---

## Disclosure

Please allow time for the issue to be investigated and fixed before public disclosure.

Responsible reporting helps keep users and hardware safe.
