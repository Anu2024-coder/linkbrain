# Code of Conduct

LinkBrain is an open-source project focused on building safe, deterministic AI-driven physical systems. We expect all contributors and community members to act professionally and respectfully.

---

## Our standards

We value:

* Respectful and constructive communication
* Technical rigor and honesty
* Safety-first engineering discussions

Unacceptable behavior includes:

* Harassment or personal attacks
* Disruptive or bad-faith contributions
* Encouraging unsafe or irresponsible use of hardware

---

## Scope

This Code of Conduct applies to all project spaces, including GitHub issues, pull requests, discussions, and community channels.

---

## Enforcement

Project maintainers may remove content or restrict participation that violates this Code of Conduct.

---

## Reporting

If you experience or witness unacceptable behavior, please contact the project maintainers privately.

By participating in this project, you agree to follow these guidelines.
